在日本考駕照攻略：台灣駕照可「免試切換」！外免切替 vs 從頭考完整比較

系列：日本生活篇｜形式：📝 文章（可搭配一支「免許中心實地流程」短片）｜目標字數：3000–5000

在日本要開車，第一個好消息給台灣人：你的台灣駕照，可以在日本免筆試、免路考直接切換成日本駕照。 這篇把兩條路——「外免切替（換照）」與「從頭到教習所考」——的流程、費用、文件與眉角一次講清楚，讓你用最省的方式拿到日本駕照。

一、先分辨你走哪條路

  • 已經有台灣駕照 → 走「外免切替（外国免許証の切替）」。台灣是日本認可的免技能與知識測驗地區之一，等於只要通過書面審查與視力檢查就能換照，是最省錢省時的路。
  • 完全沒有駕照 → 走「教習所（自動車学校）從頭考」。學科 + 技能一路上完再到免許中心考本照，費用高但最完整。

先確認自己屬於哪一種，再往下看對應段落。

二、外免切替：台灣駕照換日本駕照（重點路線）

資格條件（很重要）

  1. 持有有效的台灣駕照。
  2. 取得台灣駕照之後，曾在台灣實際居留合計滿 3 個月以上。這一點會用護照出入境章或戶籍資料佐證，是最常卡關的地方，務必備齊證明。
  3. 現在持有日本在留資格（在留卡）。

需要準備的文件

  • 台灣駕照正本
  • 駕照的日文翻譯本：可向 JAF（日本自動車聯盟） 或 台北駐日經濟文化代表處 申請
  • 護照（現用 + 舊護照，用來證明取得駕照後在台居留滿 3 個月）
  • 在留卡
  • 住民票（記載本籍/國籍，發行 3 個月內）
  • 證件照 1 張（3cm×2.4cm，規格依中心公告）
  • 申請手續費（收入印紙，依都道府縣，約數千日圓）

流程

  1. 預約：到住所所在地的運転免許センター（免許中心）預約外免切替的受理時段（多數需事前電話或線上預約）。
  2. 書面審查：確認駕照有效性、居留 3 個月證明、翻譯本。
  3. 適性檢查：視力、辨色力等基本檢查。
  4. （台灣免除）知識/技能測驗：因台灣屬認可地區，通常免筆試與路考。
  5. 當天發照：審查通過後即可領到日本駕照。
提醒：實務上各免許中心的受理時間與細節略有差異，出發前務必先打電話確認「當天要帶什麼、幾點受理、是否需預約」。

三、從頭考：教習所（自動車学校）完整流程

沒有駕照的人，得走日本正規的教習所路線。

費用

  • 普通車 MT/AT：約 25～35 萬日圓（都市較貴、鄉下較便宜）。
  • 合宿（集中住宿班）較便宜、約 2～3 週；通學班較彈性、但拉得較長。

流程

  1. 報名教習所、做適性檢查。
  2. 第一段階：學科 + 場內技能，通過後考「仮免許（臨時駕照）」。
  3. 第二段階：學科 + 路上技能（道路實際駕駛）。
  4. 修了：教習所結業。
  5. 本試験：到免許中心考學科本試驗（技能通常已由教習所認證免試），通過即發照。

給新手的建議

  • 日文不夠好？部分教習所提供英文或中文教材/口譯，報名前先問。
  • 想省錢又有時間 → 選鄉下的合宿免許，含食宿常比都市通學便宜。

四、兩條路快速比較

項目外免切替（有台灣駕照）教習所從頭考
費用數千～上萬日圓25～35 萬日圓
時間最快 1 天2 週～數月
筆試/路考台灣免除需要
關鍵門檻取得駕照後在台居留滿 3 個月的證明日文能力、時間、預算

結論很清楚：只要你有台灣駕照，一定走外免切替，別去教習所花大錢。

五、外免切替實戰補充：翻譯本與當天流程

翻譯本怎麼申請（JAF 路線最方便）

外免切替一定要附「駕照的日文翻譯本」，最常用的是 JAF（日本自動車聯盟）：

  • 受理方式：JAF 官網線上申請、郵寄，或到各地 JAF 分部臨櫃辦理。
  • 需要的東西：台灣駕照正反面影本（部分需正本核對）、申請書、費用。
  • 費用：一份約 3,000 日圓 左右。
  • 時間：線上/郵寄約需數個工作天到一週，務必提早辦，別等到預約日前一天才想到。
  • 另一選項：台北駐日經濟文化代表處也可出具翻譯/證明，時間與規費依該處公告。

外免切替當天時間表（範例）

多數免許中心把外免切替排在平日特定時段，流程大致如下，順利的話半天內完成：

  1. 09:00 報到、抽號、遞交文件
  2. 09:30 書面審查（駕照效期、居留 3 個月證明、翻譯本）
  3. 10:00 適性檢查（視力、辨色）
  4. 10:30 拍照、繳費（收入印紙）
  5. 11:30 領取日本駕照
關鍵提醒：幾乎都要事前預約，且外免切替受理時間常常「一天只有一個時段」。沒預約直接跑去，很可能白跑一趟。出發前一定打電話確認。

六、拿到日本駕照後：新手上路須知

換到照只是開始，日本開車有幾個和台灣不同、務必記住的點：

  • 靠左行駛、方向盤在右：剛開會不習慣，先在住家附近熟悉。
  • 「止まれ」（停止）要完全停：看到紅色倒三角「止まれ」標誌，輪胎必須完全靜止 1 秒再走，警察很常在這裡開單。
  • 新手標誌（初心者マーク）：在日本第一次考取者一年內須貼；外免切替者原則免貼，但建議新手自貼較安全。
  • 停車要有車位證明：買車需向警察署申請「車庫証明」。臨時停車用 coin parking（時間制收費），別違停。
  • 養車成本：每年有自動車税、每兩年「車検（しゃけん）」、強制的自賠責保險 + 建議加保的任意保険，這些都要算進預算。
  • 冬天雪地：下雪地區要換冬胎（スタッドレスタイヤ）或帶雪鏈，否則危險且可能違規。
  • ETC：上高速公路裝 ETC 較方便也常有折扣。

七、常見問題 FAQ

Q1：台灣駕照過期了還能切換嗎？
不行，必須是有效的駕照。過期請先在台灣更新後再來切換。

Q2：「居留滿 3 個月」怎麼證明？
用護照的出入境章計算取得駕照後在台停留天數；出入境章不清楚時，可向戶政或移民署申請入出國紀錄。

Q3：國際駕照可以直接在日本一直開嗎？
國際駕照（日內瓦公約）在日本最多只能用 1 年，且長期居住者有使用限制，長住還是要切換成日本駕照。

Q4：翻譯本自己翻可以嗎？
不行，必須是 JAF 或台北駐日代表處等指定單位的翻譯本才被受理。

Q5：切換要考視力，我有近視怎麼辦？
戴眼鏡/隱形眼鏡通過即可，駕照上會註記「眼鏡等」。

Q6：AT（自排）和 MT（手排）差在哪，該考哪個？
若走教習所從頭考，AT 限定較便宜、較快，日本市售車也多為自排，一般人考 AT 就夠；除非工作要開手排商用車才考 MT。外免切替則依你台灣駕照的種類對應。

Q7：切換過去的日本駕照，多久要更新一次？
日本駕照也有效期（依年齡與違規紀錄為 3 或 5 年），到期前到免許中心更新，優良駕駛（無違規）效期較長。

六、行動清單

  • 確認台灣駕照仍在有效期內
  • 備妥「取得駕照後在台居留滿 3 個月」的護照/紀錄證明
  • 向 JAF 或駐日代表處申請日文翻譯本
  • 準備在留卡、住民票、護照、證件照
  • 打電話向住所地免許中心預約外免切替、確認當天須知
  • 當天到場完成審查與視力檢查，領照

結語

日本考駕照最大的迷思，就是以為外國人都得花三十萬去教習所。事實上台灣人有免試切換的優勢，重點只在於「把文件（尤其是居留 3 個月證明）準備齊」。照這篇清單走，你很可能一天就能拿到日本駕照。

（各都道府縣免許中心的受理規則與費用可能略有不同，請以當地免許中心最新公告為準。）
